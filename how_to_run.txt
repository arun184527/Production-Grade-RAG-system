Prerequisites
 - Python (3.10+)
 - Ollama
 - ngrok
Install Dependencies
 - pip install fastapi uvicorn gradio requests faiss-cpu sentence-transformers

TERMINAL 1 — Start LLM (Ollama)
 - ollama run tinyllama  - if you see this ">>> Send a message" then  Ctrl + D

TERMINAL 2 — Start FastAPI Backend
 - uvicorn api.app:app --host 0.0.0.0 --port 8000 --reload

TERMINAL 3 - Start UI 
 -  cd ui
 -  python app.py

TERMINAL 4 — Start ngrok (Public URL)
 - ngrok http 7860 
 https://your-ngrok-url.ngrok-free.app here you see this link
